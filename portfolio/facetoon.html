FaceToon



FaceToon is a machine learning project designed to transform real-life images into user-customizable cartoon styles using Python, TensorFlow, Numpy and CycleGANs. This initiative merges artistic creativity with advanced technology, offering a distinctive way for users to turn their photos into cartoon art.

Inspired by photo-editing tools like Adobe Photoshop and Snapchat, which allow users to apply various filters to their images, the project aimed to replicate and deepen the understanding of such features through deep learning techniques.



Developed a baseline model of 100 cartoon images for initial testing. Model optimization was performed with 1500+ iterations using a mixture of 1,450 pre-processed images. Given the lack of caricature data available online.

Managed to output 16 accurate images from a random batch of 50 output images, with a model accuracy of 76%. This accuracy was reasonably good as compared to the original architecture paper, considering the time limitations and resources available.
